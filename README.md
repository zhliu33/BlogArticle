# 文章仓库说明

本仓库用于统一存放个人/团队所有博客文章的原始稿件、修订版本及相关资源，旨在实现博客内容的规范化管理、便捷归档与高效复用，同时为内容迭代、备份与分享提供可靠支撑。

## 仓库定位

作为博客内容的核心托管载体，兼顾「归档备份」与「创作管理」双重需求，既保障所有文章数据的安全存储，也方便创作者快速检索、修改稿件，同时为后续博客平台发布、内容二次传播提供基础支持。

## 内容范围

本仓库收录的内容包括但不限于：

- 博客文章原始稿件（以Markdown格式为主，兼容少量HTML格式）；

- 文章修订版本（按修改时间/版本号分类，保留修改痕迹，便于回溯）；

- 文章配套资源（如插入图片、封面图、引用文件等，统一存放于指定资源目录）；

- 文章元信息文档（记录每篇文章的发布平台、发布时间、标签、阅读量等基础信息）；

- 创作规范与模板（统一文章排版、标题格式、引用规范等，保障内容一致性）。
